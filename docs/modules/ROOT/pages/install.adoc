= Install

== Runtime baseline

Current repository baseline:

* PHP `^8.4`
* Symfony `^8.0`
* Doctrine ORM `^3.3`

== Package-oriented setup surfaces

* `composer.json`
* `config/component/services.yaml`
* `config/component/routes.yaml`
* `config/component/doctrine.yaml`
* `phpunit.xml.dist`
* `tests/Application/`
* `phpstan.neon`

== Notes

This repository is a Symfony bundle package, not a standalone application.
Root application bootstrap files are intentionally absent.
Runtime composition belongs to the host application, while package verification uses the embedded test application under `tests/Application/`.
